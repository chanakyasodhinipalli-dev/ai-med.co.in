Email Trigger & Notification Setup Guide

AI.MeD Website — Contact Form Email Configuration
For: GoDaddy Static Hosting | Zero Backend Required

Note: This guide is for the internal team. Do not publish this page on the live website.

Option A — Formspree (Recommended)

Recommended Zero backend, works directly with GoDaddy static hosting.

Setup Steps

  1. 1 Create account: Go to formspree.io and sign up for a free account.
  2. 2 Create new form: Name it AI.MeD Website Inquiries
  3. 3 Copy form ID: You'll receive a unique form ID (e.g., xyzabc12)
  4. 4 Update contact.html: Set the form action URL: 
    <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
  5. 5 Configure in Formspree Dashboard:
    • Notification email: hello@ai-med.co.in (or your business email)
    • Reply-To: set to submitter's email field {email}
    • Subject template: AI.MeD Inquiry: {inquiry_type} from {name} at {organization}
    • Enable submission email notifications (every submission)
    • Enable auto-responder to submitter
    • Enable reCAPTCHA spam protection
  6. 6 Configure auto-responder message: 
    Thank you for reaching out to AI.MeD.

We've received your inquiry regarding {inquiry_type}
and will respond within 2 business days.

In the meantime, explore our platforms at https://ai-med.co.in

— The AI.MeD Team
  7. 7 Test: Submit a test inquiry → verify email received at your inbox
Cost: Free tier = 50 submissions/month | $10/month for 1,000 submissions/month

Option B — Zapier Automation Layer

Add-On Connect Formspree to Slack, Google Sheets, and CRM.

Setup Steps

  1. 1 In Formspree: Go to Settings → Webhooks → Add your Zapier webhook URL
  2. 2 Create the following Zaps in Zapier:
Zap Trigger Action
Zap 1 New Formspree submission Send email via Gmail/Outlook with all form fields
Zap 2 New submission Post to Slack channel #website-inquiries
Zap 3 New submission Create row in Google Sheets (inquiry log with timestamp)
Zap 4 (optional) If inquiry_type = "Investment" Create contact in HubSpot CRM

Slack Message Format

New AI.MeD Inquiry

Name: {name} | Org: {organization}
Type: {inquiry_type}
Platforms: {platforms}
Email: {email}
Message: {message}

Option C — EmailJS (Alternative)

Alternative Pure frontend email — no Formspree needed.

Setup Steps

  1. Create account at emailjs.com (free: 200 emails/month)
  2. Connect your Gmail or Outlook account as an email service
  3. Create an email template with all form variables
  4. Copy your Service ID, Template ID, and Public Key
  5. Replace Formspree code in form.js with EmailJS SDK calls

EmailJS Implementation (replace Formspree fetch in form.js)

<!-- Add to contact.html head -->
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>

<script>
// Initialize EmailJS
emailjs.init('YOUR_PUBLIC_KEY');

// Replace the fetch() call in form.js with:
emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', {
  name: formData.get('name'),
  organization: formData.get('organization'),
  email: formData.get('email'),
  inquiry_type: formData.get('inquiry_type'),
  message: formData.get('message'),
  platforms: platforms.join(', '),
  role: formData.get('role'),
  phone: formData.get('phone'),
  referral_source: formData.get('referral_source')
})
.then(function(response) {
  // Show success state
  formContainer.style.display = 'none';
  formSuccess.classList.add('visible');
})
.catch(function(error) {
  // Show error state
  globalError.classList.add('visible');
});
</script>
Cost: Free tier = 200 emails/month | $10/month for 1,000 emails/month

Recommended Workflow for AI.MeD

Production Setup

  1. Primary: Use Formspree for reliability + email notification to your business inbox
  2. Slack: Use Zapier to simultaneously post to a Slack channel #website-inquiries
  3. Backup CRM: Store all submissions in Google Sheets via Zapier as a backup inquiry log
  4. Monthly review: Review all inquiries, follow up on non-responses, update pipeline tracking

Testing Checklist

  • ☐ Submit test inquiry from contact.html
  • ☐ Verify notification email arrives at business inbox
  • ☐ Verify auto-responder email arrives at submitter's inbox
  • ☐ Verify email subject line includes inquiry type and name
  • ☐ Verify Slack notification posts correctly (if Zapier connected)
  • ☐ Verify Google Sheets row is created (if Zapier connected)
  • ☐ Test form validation (empty fields, invalid email, short message)
  • ☐ Test success state display after submission
  • ☐ Test error state display (disconnect internet and submit)
  • ☐ Test on mobile device
  • ☐ Test reCAPTCHA spam protection
